## ğŸš€ Instalasi

### Persyaratan
- [Node.js](https://nodejs.org/) v18.0.0 atau lebih tinggi
- [Git](https://git-scm.com/)
- RAM minimal 1GB

### Langkah Instalasi

```bash
# Clone repository
git clone https://github.com/PanzzDevv/panzzbot.git

# Masuk ke direktori
cd panzzbot

# Install dependencies
npm install

# Jalankan bot
npm start
```

## âš™ï¸ Konfigurasi

Edit file `config.js` untuk konfigurasi bot:

```javascript
owner: {
    name: 'YourName',
    number: ['628xxxxxxxxxx']  // Format: 628xxx
},

session: {
    pairingNumber: '628xxxxxxxxxx',  // Nomor WA untuk di-pair
    usePairingCode: true             // true = Pairing Code
},

bot: {
    name: 'PANZZBOT',
    version: '1.0.0',
    developer: 'PANZZ'
},

command: {
    prefix: '.'  // Prefix command (.menu, .help, dll)
}
```
